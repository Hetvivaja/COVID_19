# COVID-19 Dashboard

A Flask dashboard that shows COVID-19 charts using local CSV files from the `data` folder.

## How to run

1. Create and activate a virtual environment.
2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Start the app:

```bash
python app.py
```

4. Open `http://127.0.0.1:8001` in your browser.
